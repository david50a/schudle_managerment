# Schedule Management

A C++ project for managing and organizing schedules efficiently.

## Project Overview

This is a schedule management application built primarily in C++ with supporting build configuration files.

## Technology Stack

- **C++** - 76.1% (Main language)
- **Makefile** - 18% (Build automation)
- **C** - 3.1% (Support code)
- **CMake** - 2.8% (Build configuration)

## Getting Started

### Prerequisites

- C++ compiler (C++11 or later)
- Make
- CMake (optional, for advanced builds)

### Building

To build the project:

```bash
make
```

Or using CMake:

```bash
mkdir build
cd build
cmake ..
make
```
